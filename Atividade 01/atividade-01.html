Descrição de algumas tags

Table

Essa tag define uma tabela em html usando também outras tags: th, tr e td;

Th

Essa tag define um cabeçalho da tabela, sendo por padrão um texto em negrito e centralizado;

Tr

Essa tag define uma linha de uma tabela;

Td

Essa tag define um elemento dentro da tabela, são onde ficam as informações da tabela. Eles
podem conter qualquer informação sendo texto, imagem, lista, etc.;

THEAD, TBODY e TFOOT

Essas tags tem como função manter a semântica das tabelas, não são obrigatórias porém ajudam
a deixa o código coerente. Sendo o thead, tbody e tfoot usados para representar o cabeçalho,
o corpo e o rodapé da tabela respectivamente;

ALIGN

É um atributo usado para especificar o alinhamento do texto de um determinado elemento.
Podendo ter os valores: left | right | center | justify;

BORDER

É uma atributo que adiciona uma borda em determinado elemento.

CAPTION

É a tag que define o título de uma tabela;

ROWSPAN

É uma atributo que define o número de linhas que determinado elemento da tabela ocupara;

COLSPAN

É uma atributo que define o número de colunas que determinado elemento da tabela ocupara;

WIDTH

É uma atributo que define a largura de um determinado elemento;
